# Saper
Classic Windows Saper Game written in JavaScript (ES5+) with classic design written in CSS.


## Built With
* JavaScript (ES5+)
* HTML
* CSS
* [SVG SILH](https://svgsilh.com/) - Used to get some icons
* [1001 Fonts](https://www.1001fonts.com/) - Used to get fonts
